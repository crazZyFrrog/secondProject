# 🚀 Quick Start - Landing Constructor

## 📁 Структура проекта

```
secondProject/
├── frontend/          # React приложение
├── backend/           # API (в разработке)
└── README.md
```

## ⚡ Быстрый запуск

### 1. Frontend

```bash
cd frontend
npm install
npm run dev
```

Откройте: **http://localhost:5173/**

### 2. Backend (в разработке)

```bash
cd backend
# Будет добавлено позже
```

## 📋 Основные команды

### Frontend

```bash
# Разработка
npm run dev

# Сборка
npm run build

# Превью сборки
npm run preview

# Проверка TypeScript
npx tsc --noEmit
```
